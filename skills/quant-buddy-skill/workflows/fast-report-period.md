# 快速执行 · 最近报告期财务

> **适用范围**：≤3 只 A 股，查询最近一个已披露报告期的财务字段（利润表/资产负债表/ROE 等）。  
> 进入本文件前必须已满足 SKILL.md 快速路径触发条件。  
> ⚠️ 财务数据仅支持 A 股。港股/美股财务查询需主动告知用户不支持。

---

## 执行步骤

```
① newSession
→ ② confirmMultipleAssets（资产数 ≥ 2 强制；单资产也建议执行）
→ ③ [字段分类]
     · 字段在下方内联模板中（含派生字段）→ 原样复制模板中引号内的完整字段名，不得增删括号后缀，跳过 confirmDataMulti
     · 字段不在模板且名称不确定          → 调用 confirmDataMulti 确认（返回空 → 退出快速路径，回到 SKILL.md 走完整探索链路）
→ ④ runMultiFormula（所有基础字段合并为一次调用）
     · 若返回 500 或 PARTIAL_SUCCESS（有字段失败）→ 对失败字段单独调用 confirmDataMulti 重新确认正确字段名，再重试该字段
     · 禁止在此步骤调用 searchSimilarCases 作为回退
→ ⑤ readData(mode="last_valid_per_asset")（必须执行，不可用 description 代替）
→ ⑥ 如有派生指标（资产负债率/毛利率），用 readData 返回的原始末值手动计算
→ ⑦ 输出最终答案（首句必须是资产名 + 报告期 + 数据结论）
```

**停止条件**：所有字段均已通过 readData 获得末值 + 末日期（= 报告期）→ 立刻停止。

## 字段公式模板

| 字段 | 公式（「资产名」= confirmMultipleAssets 返回的标准名） |
|------|------|
| 营业收入 | `{资产名}_营收 = "A股营业收入〔报告期利润表〕" * 取出(资产名)` |
| 净利润 | `{资产名}_净利润 = "A股净利润〔报告期利润表〕" * 取出(资产名)` |
| 归母净利润 | `{资产名}_归母净利润 = "A股归属于母公司股东的净利润〔报告期利润表〕" * 取出(资产名)` |
| 营业成本 | `{资产名}_营业成本 = "A股营业成本〔报告期利润表〕" * 取出(资产名)` |
| 总资产 | `{资产名}_总资产 = "A股资产总计〔报告期资产负债表〕" * 取出(资产名)` |
| 总负债 | `{资产名}_总负债 = "A股负债合计〔报告期资产负债表〕" * 取出(资产名)` |
| 净资产 | `{资产名}_净资产 = "A股股东权益合计〔报告期资产负债表〕" * 取出(资产名)` |
| ROE | `{资产名}_ROE = "A股净资产收益率ROE" * 取出(资产名)` |
| 净利率（单季） | `{资产名}_净利率 = "A股销售净利率：单季〔财务指标〕" * 取出(资产名)` |
| 经营活动现金流净额 | `{资产名}_经营现金流 = "A股经营活动产生的现金流量净额〔报告期现金流量表〕" * 取出(资产名)` |

**派生指标（直接构造，无需 confirmDataMulti）**：

| 派生字段 | 构造方式 | 所需基础公式 |
|------|------|------|
| 资产负债率 | readData 后计算：总负债 ÷ 总资产 × 100 | 总负债 + 总资产 |
| 毛利率 | readData 后计算：(营业收入 - 营业成本) ÷ 营业收入 × 100 | 营业收入 + 营业成本 |

> 派生指标**不在数据平台字段库中**，直接用基础字段公式获取原始值，readData 后手动计算，不要调用 confirmDataMulti 查询派生字段名。

> ⚠️ 变量名必须加资产名前缀（`{资产名}_指标名`），否则触发循环依赖 500 错误。

**runMultiFormula 固定参数**：
```json
{"begin_date": 20230101, "include_description": true, "use_minute_data": true}
```

---

## 报告期一致性（多字段必须执行）

多字段时，确认所有字段 readData 返回的末日期一致后才合并回答。

- 日期一致 → 正常合并
- 日期不一致 → 声明「字段披露期不一致」，分别报告各字段及其对应期，不得混算派生指标（资产负债率、毛利率等）

---

## 取值规则

- 财务末值和报告期**必须来自 readData 返回**，不得从 description 直接推断
- 涉及派生计算（资产负债率 = 总负债/总资产）时，必须基于 readData 原始数值计算
- 百分比字段（ROE、净利率）返回小数时 ×100 加 `%`

---

## 保护规则（4 条）

1. **evidence-only**：只输出 readData 末值；禁止推断归因或补充扩展信息
2. **去过程化**：首句必须是资产名 + 报告期 + 数据结论；禁止「已成功获取」「数据已获取」等话术
3. **条件冻结**：用户条件原样执行，百分比不转小数
4. **报告期声明**：回答中必须明确标注报告期（如 `2025-12-31`），不得仅写"最新"
