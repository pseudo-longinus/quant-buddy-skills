# 快速执行 · 最新时点行情/估值快照

> **适用范围**：≤3 个资产，查询最新交易日的行情/估值少量字段（标量），立刻回答。  
> 进入本文件前必须已满足 SKILL.md 快速路径触发条件（条件不满足请走完整链路）。

---

## 执行步骤（5 步，严格顺序）

```
① newSession
→ ② confirmMultipleAssets（资产数 ≥ 2 强制；单资产也建议执行）
→ ③ runMultiFormula（所有字段合并为一次调用）
→ ④ 从 description 末值 + 末日期提取结果
→ ⑤ 输出最终答案（首句必须是资产名 + 数据结论）
```

**停止条件**：所有字段均已获得末值 + 末日期 → 立刻停止，禁止继续调用任何工具。

---

## 字段公式模板

> ⚠️ 估值字段（PE/PB/总市值/换手率）仅适用于 A 股。

| 字段 | 公式（「资产名」= confirmMultipleAssets 返回的标准名） |
|------|------|
| 收盘价 | `{资产名}收盘 = "全市场每日收盘价" * 取出(资产名)` |
| 开盘价 | `{资产名}开盘 = "全市场每日开盘价" * 取出(资产名)` |
| 涨跌幅 | `{资产名}涨幅 = "全市场每日回报率" * 取出(资产名)` |
| 成交额 | `{资产名}成交额 = "全市场每日成交额" * 取出(资产名)` |
| 成交量 | `{资产名}成交量 = "全市场每日成交量" * 取出(资产名)` |
| PE（TTM） | `{资产名}PE = "A股市盈率（PE, TTM）〔估值数据〕" * 取出(资产名)` |
| PB | `{资产名}PB = "A股市净率（PB）〔估值数据〕" * 取出(资产名)` |
| 总市值 | `{资产名}总市值 = "A股总市值〔估值数据〕" * 取出(资产名)` |
| 换手率 | `{资产名}换手率 = "A股换手率〔估值数据〕" * 取出(资产名)` |

**字段不在上表时**：调用 `confirmDataMulti` 确认数据集名称，再构造公式；返回空/不支持 → 告知用户，停止执行。

**runMultiFormula 固定参数**：
```json
{"begin_date": 20240101, "include_description": true, "use_minute_data": true}
```

**多资产示例**：
```json
{"formulas": [
  "比亚迪收盘 = \"全市场每日收盘价\" * 取出(比亚迪)",
  "比亚迪涨幅 = \"全市场每日回报率\" * 取出(比亚迪)",
  "宁德时代收盘 = \"全市场每日收盘价\" * 取出(宁德时代)",
  "宁德时代涨幅 = \"全市场每日回报率\" * 取出(宁德时代)"
], "begin_date": 20240101, "include_description": true, "use_minute_data": true}
```

---

## 取值规则

- 从每个公式对应的 `description` 取**最后值**（末行数值）和**末日期**
- 涨跌幅为小数（如 `-0.0274`）→ ×100 再加 `%`（`-2.74%`）
- 若末日期早于当前交易日，声明"以下为最后可得交易日 YYYY-MM-DD 的结果"

---

## 保护规则（4 条）

1. **evidence-only**：只输出 description 末值；禁止推断归因或补充扩展信息
2. **去过程化**：首句必须是资产名 + 数据结论；禁止「已成功获取」「让我来」等话术
3. **条件冻结**：用户条件原样执行，百分比不转小数
4. **涨跌幅换算**：小数收益率 ×100 再加 %
